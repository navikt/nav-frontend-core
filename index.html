NAV-frontend-core v. 2 inneholder globale variabler.

Lenker: Den inneholder ikke lenger global styling av alle a-elementer. Det tilbys i stedet følgende:

  • Mixin .mixin-lenke(), som gir riktig lenkestyling.
  • Klassen .lenke, som kan extendes og/eller brukes på a-elementer. Extending med alle tilstander gjøres slik i less: 
                        .brodsmuler a {
                        &:extend(.lenke all);
                    }

    Output blir som følger:

                        .lenke, .brodsmuler a {
                        color: #0067c5
                        /* etc */
                    }
